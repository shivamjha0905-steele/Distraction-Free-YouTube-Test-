# 🎯 Distraction-Free YouTube (HTML Only)

## 📖 Overview
This project is a simple **YouTube viewer page** built using **only HTML**.  
It removes clutter such as sidebars, recommendations, and comments — allowing users to focus solely on the video content.

## ✨ Features
- Clean, distraction-free layout  
- Embedded YouTube video using `<iframe>`  
- Lightweight: no CSS or JavaScript used  
- Beginner-friendly HTML structure  

## 🛠️ Tech Stack
- **Language:** HTML  
- **No frameworks/libraries** — pure fundamentals  

## 📂 How to Use
1. Clone or download the repository.  
2. Open the `index.html` file in any browser.  
3. Replace the YouTube video link inside the `<iframe>` with your preferred video.  

```html
<iframe width="560" height="315" 
src="https://www.youtube.com/embed/VIDEO_ID" 
frameborder="0" allowfullscreen></iframe>
